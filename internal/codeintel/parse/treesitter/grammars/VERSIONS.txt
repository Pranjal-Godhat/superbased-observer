Embed model (W1 relief valve, ADR-0006): build.sh emits raw codeintel_<lang>.wasm
then zstd-compresses each to codeintel_<lang>.wasm.zst (zstdpack.go). Only the
*.wasm.zst is committed + go:embed'd (decompressed in-memory at runtime); the
raw *.wasm is gitignored build output.

The harness (extract.c) evaluates the standard text predicates #eq? / #not-eq?
/ #any-of? / #not-any-of? itself (the low-level query cursor does not), so a
query like the JS/Lua require() rule or the Ruby require/require_relative rule
can gate on @capture text.

Core tier (thorough calls): rust/typescript/tsx/python/javascript/jsx + java/
c/cpp/csharp/ruby/php. Extended tier (S + best-effort calls): kotlin/swift/
scala/bash/lua. Scala imports are intentionally not captured (flat per-segment
path fields); bash has no import construct. Go is served exactly by goast (no
wasm grammar). ALL 18 v1 languages now have exact spans. Adding another is
data: see docs/codeintel/adding-a-language.md.
